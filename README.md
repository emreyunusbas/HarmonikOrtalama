### Proje Açıklaması

Bu program, kullanıcının girdiği sayıların harmonik ortalamasını hesaplar. Kullanıcıdan kaç tane sayı gireceği istenir, ardından girilen sayıların harmonik serisi hesaplanır ve harmonik ortalama bulunur.

---

### Dosyalar

Bu repo aşağıdaki dosyaları içermektedir:

- `README.md`: Proje hakkında genel bilgiler ve açıklamalar içeren dosya.
- `HarmonikOrtalama.java`: Programın kaynak kodunu içeren Java dosyası.

---

### Nasıl Kullanılır

1. Proje dosyalarını bilgisayarınıza indirin veya kopyalayın.
2. Java IDE'si (Integrated Development Environment) veya metin düzenleyici gibi bir yazılım kullanarak `HarmonikOrtalama.java` dosyasını açın.
3. Kaynak kodunu inceleyin ve gerekirse düzenlemeler yapın.
4. Düzenlemeleri tamamladıktan sonra, `HarmonikOrtalama.java` dosyasını derleyin ve çalıştırın.
5. Program size kaç tane sayı gireceğinizi soracaktır.
6. İstenilen sayıları girin ve program girilen sayıların harmonik ortalamasını hesaplayacaktır.
